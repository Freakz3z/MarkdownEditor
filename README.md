# 基于Swing的简易Markdown编辑器
![image](https://github.com/user-attachments/assets/2b122f39-1e2c-438b-93a8-ee1f470b5c83)

## 功能
- 支持使用Markdown的语法编写
- 保存上一次关闭编辑器时的输入于`temp.txt`
- 支持输出`.txt` `.md` `.html`文件
- 导出后的HTML文件已通过CSS美化

## 不支持的语法
1. 分隔线：`---`、`***`、`___`
2. 图片：`![]()`
3. 引用：`>`
4. 表格
5. 无序列表的子项目


## 已知Bug
- 在编辑器的HTML预览界面点击链接无法跳转
